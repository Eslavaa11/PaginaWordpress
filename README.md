# Proyecto Web en WordPress

## Descripción

Este proyecto corresponde al desarrollo de una página web realizada en WordPress, utilizando una plantilla visual y 
herramientas de edición como Elementor/Astra. El objetivo principal fue construir una página funcional, organizada 
y visualmente agradable, aplicando los conocimientos vistos durante el proceso de desarrollo web.

## Estado del proyecto

La página fue desarrollada localmente/en entorno de WordPress; sin embargo, no fue posible publicarla en un dominio 
de WordPress en línea debido a limitaciones técnicas presentadas durante el proceso de despliegue.

## Justificación

Durante el desarrollo se intentó subir y configurar la página en un dominio o servidor compatible con WordPress, 
pero se presentaron inconvenientes relacionados con la publicación del sitio, la configuración del hosting y la 
compatibilidad necesaria para que WordPress funcionara correctamente en línea.

Debido a esto, no se logró completar la publicación en un dominio web dentro del tiempo disponible, pero el 
proyecto fue trabajado y estructurado en WordPress, cumpliendo con el objetivo principal de diseño y construcción 
del sitio.

## Dificultades encontradas

- Problemas para encontrar un hosting gratuito completamente compatible con WordPress.
- Limitaciones para publicar el sitio en GitHub Pages, ya que WordPress no funciona como una página estática.
- Inconvenientes con plugins y plantillas durante el proceso de edición.
